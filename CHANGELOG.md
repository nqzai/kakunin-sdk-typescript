# Changelog

All notable changes to this project are documented here. The format is based on

## [0.2.0] — 2026-07-07
### Added
- Public release of `@kakunin/sdk` (Apache-2.0), published to npm with provenance.
- Resources: agents, events, certificates, verify, webhooks; static `computeModelHash`.
- `@kakunin/sdk/verify` enforcement middleware and `@kakunin/sdk/verify/supabase`
  RLS binding (`bindAgentSession`).
